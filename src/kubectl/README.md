# kubectl (via official release channel) (kubectl)

Kubectl is used to manage a kubernetes cluster

## Example DevContainer Usage

```json
"features": {
    "ghcr.io/jojoxd/devcontainers/kubectl:1": {}
}
```

## Options

| Options Id | Description | Type | Default Value |
